01
Linear Regression
Establishes a linear relationship between features and the continuous target variable (Item_Outlet_Sales). The model computes optimal coefficients by minimising the sum of squared residuals (OLS method).

Equation: ŷ = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

Appropriate when a linear relationship exists between predictors and outcome. Interpretable — each coefficient quantifies the marginal effect of a feature.
R² SCORE 0.503
02
Logistic Regression
Despite its name, Logistic Regression is a classification algorithm. It applies the sigmoid function to model the probability of a sample belonging to a particular class.

Applied as: Sales categorised into Low / Medium / High. The model predicts which category a given product-outlet combination belongs to.

Output is a probability between 0 and 1, converted to a class label via a decision threshold.
ACCURACY 67.1%
03
SVR (RBF Kernel)
Support Vector Regression finds an optimal hyperplane within an ε-insensitive tube. Only data points outside this tolerance band (support vectors) influence the model.

RBF Kernel: Projects data into higher dimensions to capture non-linear relationships — critical for this dataset where sales does not scale linearly with MRP.

Parameters: C=100 (regularisation), ε=0.1 (tolerance margin).
R² SCORE 0.603 ★
